# Sistema de Gestión Electoral UTN.BA

Maquetado HTML del módulo **PRE-VOTACIÓN** para el proceso de Junta Electoral.

## Usuario del prototipo

- Usuario: Juan Pérez
- Proceso: Junta Electoral

## Pantallas incluidas

- Inicio
- Nueva elección
- Configuración de parámetros
- Elecciones en curso
- Elecciones pasadas

## Cómo abrir el prototipo

Abrir el archivo `index.html` en un navegador.

## Estructura del repositorio

```text
sistema-gestion-electoral-utn/
```

## Criterio visual

La interfaz mantiene un estilo institucional inspirado en UTN.BA, usando azul como color principal operativo, rojo como acento institucional, fondos claros, tarjetas, tablas y botones administrativos.
